Interfacepublic interface IUrlApplicationState extends flash.events.IEventDispatcher
SubinterfacesIUrlRule

Interface for a general representation of an application's state in terms of a URL and a title. The application state is exposed via the url and title read-only properties, and may be set from whatever container is managing the application by setting the containerURL property (typically an instance of UrlBrowserManager).

State changes initiated by the application will be signaled by dispatching the stateChange event.
